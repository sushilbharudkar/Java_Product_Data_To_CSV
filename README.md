# Java Product Data to CSV

## Description
This Java mini project stores product details such as name, price, and rating
and exports the data into a CSV file using file handling.

## Key Features
- Stores product information
- Writes data into CSV format
- Uses FileWriter class
- Demonstrates Java file handling
- Simple and readable code

## Tech Stack
- Java
- File Handling
- FileWriter class

## How to Run
1. Clone or download the repository
2. Compile the program:
   javac ProductDataToCSV.java
3. Run the program:
   java ProductDataToCSV

## Purpose
- Learn file handling in Java using FileWriter
- xport data into CSV format for real-world applications
- Develop beginner-friendly data storage and export mini projects
- This project demonstrates basic file handling operations in Java for internship practice.
